Privatlivspolitik

Sidst opdateret: marts 2026

Introduktion

AS-drive ("appen") er en enterprise-applikation til billedoptagelse og upload til SharePoint. Denne politik beskriver, hvordan vi behandler data.

Dataindsamling

Appen indsamler og behandler følgende data:

  • Billeder: Fotos taget med appens kamera gemmes midlertidigt på enheden og uploades til jeres virksomheds SharePoint. Billeder gemmes IKKE i telefonens fotobibliotek.
  • ID-numre: Numeriske identifikatorer som brugeren indtaster for at organisere billeder i mapper.
  • Microsoft-konto: Appens login-oplysninger håndteres af Microsoft Authentication Library (MSAL). Vi gemmer ikke adgangskoder.
  • Konfiguration: TenantID, ClientID, SiteID og DriveID gemmes lokalt på enheden.

Datahåndtering

  • Billeder krypteres med iOS' hardware-baserede kryptering (Complete File Protection) mens enheden er låst.
  • Billeder slettes automatisk fra enheden umiddelbart efter succesfuld upload (Zero-Persistence).
  • Ingen data sendes til tredjeparter. Al kommunikation foregår mellem enheden og jeres virksomheds Microsoft 365/SharePoint-miljø.
  • Appen bruger udelukkende Microsoft Graph API til dataoverførsel.

Tredjepartstjenester

Appen bruger følgende tredjepartstjenester:

  • Microsoft Authentication Library (MSAL): Til sikker login med Microsoft-konto. Se Microsofts privatlivspolitik.
  • Microsoft Graph API: Til upload af filer til SharePoint.

Datalagring

Appen gemmer ingen permanente brugerdata. Konfigurationsdata (ikke følsomme) gemmes i UserDefaults på enheden. MSAL tokens gemmes sikkert i iOS Keychain.

Brugerrettigheder

Du kan til enhver tid:

  • Logge ud af din Microsoft-konto i appen
  • Slette appen og al tilknyttet data fra enheden
  • Kontakte din IT-administrator for at tilbagekalde app-adgang
