# Hejmadi.com - Dykkermedicin Kursus Underside

## Filstruktur

Upload følgende filer til din Hejmadi.com webserver:

```
hejmadi.com/
├── index.html                    ← Erstat din nuværende index.html
├── dykkermedicin-kursus.html     ← Ny underside
├── images/
│   ├── dykker-hero.jpg           ← Hero-billede til video poster
│   ├── dykker-undervand.jpg      ← Dykker under vand
│   ├── trykkammer-interior.jpg   ← Indvendigt i trykkammer
│   ├── trykkammer-eksterior.jpg  ← Eksternt trykkammer
│   ├── undervisning-1.jpg        ← Undervisningslokale
│   ├── undervisning-2.jpg        ← Forelæsning
│   ├── spirometri.jpg            ← Spirometri-undersøgelse
│   ├── otoskopi.jpg              ← Øreundersøgelse
│   └── dykkermedicin-preview.jpg ← Preview til index.html
└── media/
    └── dykkermedicin-intro.mp4   ← Introduktionsvideo
```

## Ændringer i index.html

1. **Navigation**: Tilføjet "Dykkermedicin" link i hovednavigationen
2. **Ny sektion**: Fremhævet boks med dykkermedicin-kurset inden kontakt-sektionen

## dykkermedicin-kursus.html

Ny underside med:
- Hero-sektion med video øverst (responsivt)
- Billeder integreret i indholdet
- Fuldt responsivt design til mobil og desktop
- Matcher Hejmadi.com's styling og farver

## URL-struktur

Undersiden vil være tilgængelig på:
- https://hejmadi.com/dykkermedicin-kursus.html

## Bemærkninger

- Billederne er gemt som .jpg selvom originaler var .png (for kompatibilitet)
- Video i MP4 format (H.264)
- Alle links og navigation er relative, så de virker på produktionsserveren
